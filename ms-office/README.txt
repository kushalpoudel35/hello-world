Step 1: Create a temporary folder on desktop, for example "office".
Step 2: Run the office deployment tool and extract the files to the folder "office".
Step 3: Mount the Microsoft office .img disc image file.
Step 4: Copy the Configuration.xml file to the "office" folder, and edit it based on what apps you want installed.
Step 5: Open powershell window in "office" folder.
Step 6: Run "./setup.exe /configure Configuration.xml".
Step 7: That's it! Done!
